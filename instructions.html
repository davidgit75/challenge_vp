Instructions (minumun required)

The following instructions should be develop in React. We have already a bolierplate project set and ready for you to develop, check how to start it on installation setup Each of the task will measure the coding skills, so it would very advisable to work with a clean, intuitive and rehusable code in order to assign you the seniority level. There's a file called src/data/data.js that represent the data you will show on the table

  1. Convert the existing html table above into react Component(s).
  2. Create an additional column within the table which adds a "delete" button to each row. Then, set up a event listener so that when a user clicks the button, it removes the row.
  3. Set up an event listener so that when a user clicks a particular row within the contact list, the "First Name", "Last Name", and "Phone" are output as a space delimited string to the existing span#result.
  4. Set up an event listener so that when a user clicks a particular cell within the table body, the cell position is output as a number to the existing span#result.
    Note: Count the cells from left to right, top to bottom.
  5. Set up an event listener so that when a user clicks a particular row within the contact list, the last four digits of the row's "Phone" are taken and subtracted from left to right (Ex: 4211 = 0, 6311 = 1). The result of the subtraction is then compared against the row number to determine if the result is greater than the row number.
    Write the boolean result of this comparison to span#result

Instructions (for react heros optional)

This is a plus, do it if you consider that you have the skills to do it, this will determine also your seniority

  1. Manage the data and the states of the app with Redux (don't use this.state in react), only props are allowed
  2. After you refresh the page the data should be available after 1 second, imitating a fetching on the server. This action should be performed async (suggesttion - react-thunk), check the method fakeFetchListData() on src/actions/index.js

Installation set up

To install and run this project you have to install Node, if you don't have it please download the lastest version here

  1. Open the bash/terminal of your preference and go to the project root
  2. Install the dependencies, write on bash/terminal npm install
  3. Run the project, write npm start on write on bash/terminal
  4. Open your browser on localhost:3000 and you will see a message
  5. To start developing the exercise go to src/Container.js and start coding
  6. Have fun
